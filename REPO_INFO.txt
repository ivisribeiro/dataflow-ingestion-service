🚀 COMO USAR:
  1. git clone https://github.com/ivisribeiro/dataflow-ingestion-service.git
  2. cd dataflow-ingestion-service
  3. cp .env.example .env
  4. docker-compose up -d
  5. Acesse http://localhost:8000
